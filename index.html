📖 项目简介

WuziQ 是一个用纯 Python 实现的极简五子棋游戏。项目支持多种界面选择：

  • 命令行界面 (CLI)：传统的终端游戏体验
  • Web 界面：基于 Flask 的轻量级网页版本
  • 桌面 GUI：使用 Tkinter 的图形化界面
  • Windows 可执行文件：开箱即用的独立应用

游戏默认采用 15×15 的棋盘，支持两人对战或与 AI 对手游玩。所有游戏逻辑仅依赖 Python 标准库实现。

🚀 快速开始

安装

核心游戏逻辑仅需 Python 标准库。如需使用 Web 界面，请安装依赖：

pip install -r requirements.txt

运行 Web UI

启动轻量级 Flask 服务器，然后在浏览器中打开 http://127.0.0.1:8000：

python -m src.webapp

服务器绑定到 0.0.0.0:8000，适合在 Codespaces 或远程开发环境中使用。 页面允许通过点击单元格放置棋子，支持悔棋、重置棋盘和更改 AI 方位。

运行桌面 UI

基于 Tkinter 的桌面客户端提供离线体验，支持鼠标输入、悔棋控制和 AI 方位切换：

python -m src.ui_tk --size 15 --ai white

WuziQ 桌面界面展示

Windows 独立可执行文件

在 Windows 11 上运行构建脚本，生成独立的 WuziQ.exe：

scripts\build_win.bat

同样的步骤也在 CI 中自动执行，构建的二进制文件会作为工件上传。

运行命令行版本

启动终端对战：

python -m src.cli

或者：

python src/cli.py

可选参数：

  • --size N — 更改棋盘尺寸（最小 5）
    python -m src.cli --size 10
  • --ai {off,black,white} — 启用 AI 对手
    black 让 AI 先手，white 让 AI 后手，默认为 off（两人模式）

游戏中的命令

游戏过程中输入棋子坐标（如 H8 或 8 7），使用以下命令：

:undo

悔棋。在 AI 对战中会一起悔回 AI 的移动，保持回合秩序。

:save filename.json

保存当前游戏状态到文件。

:load filename.json

加载已保存的游戏。

:quit

退出游戏。

✨ 主要特性

多种界面

支持命令行、Web 和桌面 GUI 三种游戏界面，满足不同需求。

智能 AI

内置简单但有效的 AI 对手，可调节 AI 方位为黑棋或白棋。

游戏存档

支持保存和加载游戏状态，随时暂停继续游戏。

自定义棋盘

灵活的棋盘尺寸设置，最小 5×5，满足各类玩法需求。

纯 Python 实现

核心逻辑仅依赖标准库，轻量级且易于扩展。

跨平台支持

支持 Windows、macOS、Linux，包括独立可执行文件。

🧠 设计架构

  • 棋盘表示：GomokuBoard 数据类封装了 2D 列表， 存储可选的玩家标识符（"B" 或 "W"）
  • 移动解析：parse_move 函数支持国际象棋风格 （H8）和数字坐标（8 7）两种输入
  • 胜利判定：check_victory 在每次移动后检查四个方向 （水平、竖直、两条对角线）的连续棋子。5 个或更多连续棋子即判定为胜利
  • 命令行循环：CLI 在玩家之间交替进行，每次有效移动后打印棋盘， 当某方获胜或棋盘填满时退出
